# 🔐 Password Manager

A secure and user-friendly Password Manager built to store, manage, and organize passwords efficiently.

## 🚀 Features

- Add new passwords
- View saved passwords
- Edit existing passwords
- Delete passwords
- Copy password to clipboard
- Responsive design
- Secure password storage

## 🛠️ Tech Stack

- React.js
- Tailwind CSS
- Node.js
- Express.js
- MongoDB
